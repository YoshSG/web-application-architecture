Infraestructura y Despliegue

Las aplicaciones se despliegan utilizando contenedores Docker orquestados con Kubernetes, aprovechando la infraestructura de nubes como AWS o GCP. Además, el pipeline CI/CD se automatiza con herramientas como Jenkins o GitLab CI.

Diagrama de Infraestructura

Pipeline CI/CD Automatizado

Un pipeline CI/CD permite automatizar el proceso de construcción, prueba y despliegue de aplicaciones. A continuación se muestra un flujo típico usando Jenkins:

  • 1. Commit de código
  • 2. Compilación automática
  • 3. Pruebas unitarias
  • 4. Despliegue en staging
  • 5. Pruebas de integración
  • 6. Despliegue en producción

Software a utilizar


Primera opción: Ubuntu Server 22.04 LTS

    Ubuntu Server es una distribución Linux ampliamente utilizada para servidores debido a su seguridad, estabilidad y rendimiento. La versión LTS (Long-Term Support) asegura soporte por 5 años, lo cual es ideal para un proyecto a largo plazo.


Ventajas:

  • Software libre y de código abierto, sin costos de licencia, y con una comunidad extensa de soporte.

Entorno de Desarrollo Integrado (IDE): Visual Studio Code

    Visual Studio Code es un IDE ligero, multiplataforma y altamente personalizable. Ofrece extensiones para múltiples lenguajes de programación y herramientas que facilitan el desarrollo.

Ejemplo de Script para CI/CD con Jenkins

Este script de Jenkinsfile automatiza el pipeline de despliegue de una aplicación web:


pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                sh 'npm install'
            }
        }
        stage('Test') {
            steps {
                sh 'npm test'
            }
        }
        stage('Deploy') {
            steps {
                sh 'kubectl apply -f deployment.yaml'
            }
        }
    }
    post {
        success {
            echo 'Despliegue completado exitosamente'
        }
        failure {
            echo 'El despliegue ha fallado'
        }
    }
}

Segunda opción: Windows Server 2022

    Windows Server es una opción sólida para entornos que ya están familiarizados con la suite de herramientas de Microsoft.


Ventajas:

  • Amplio soporte empresarial y fácil integración con otras soluciones de Microsoft.

Entorno de Desarrollo Integrado (IDE): Microsoft Visual Studio

    Visual Studio es una potente herramienta de desarrollo que ofrece integración directa con plataformas como Azure y SQL Server.


Lenguaje de Programación: C# con el framework ASP.NET Core

    ASP.NET Core es un framework de alto rendimiento y multiplataforma, optimizado para aplicaciones modernas basadas en la nube.
